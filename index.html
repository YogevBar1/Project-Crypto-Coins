    About

    My name is Yogev Bar, and this is my second project in the John Bryce Full Stack course. This project showcases my web development skills and knowledge gained throughout the course.

    This crypto project is a web application that allows users to explore and track information about various cryptocurrencies. The application retrieves data from an API and displays it in a user-friendly interface. Users can view a list of cryptocurrencies, toggle their selection, and obtain additional information about each coin, including its image, current price in USD, EUR, and ILS. The application also supports searching for specific coins by name. Additionally, the project includes features such as storing selected coins in local storage and a modal dialog for managing the selection of up to five coins. Overall, this project provides a convenient way for users to stay informed about cryptocurrency market data and manage their preferred coin selection.
